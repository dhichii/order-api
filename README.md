<h1 align="center">Order API</h1>

## Getting Started

### Prerequisites
- Go 1.8+
- PostgreeSQL 14.+

### Installation or Configure
- Clone the repo
```bash
# clone if you don't have the code base
$ git clone git@github.com:dhichii/order-api.git

# tidy up and run
$ go mod tidy
```
- Setup your env\
copy the `example.env`, rename to `app.env`, and define your own env
- run the app
```bash
$ go run main.go
```
